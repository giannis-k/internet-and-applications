# CTGOV-02: Συχνά Φάρμακα για την αντιμετώπιση μιας συγκεκριμένης Ασθένειας

Δικτυακή εφαρμογή η οποία βρίσκει τα 10 (max) πιο συχνά φάρμακα που έχουν χορηγηθεί για την αντιμετώπιση μιας συγκεκριμένης ασθένειας. Το input θα είναι το όνομα της ασθένειας το οποίο θα το ορίζει ο χρήστης μέσω της web σελίδας και το output θα είναι τα 10 (max) πιο συχνά φάρμακα (για την ακρίβεια, φαρμακευτικές ουσίες) καθώς και η συχνότητα χρήσης τους. Τα δεδομένα θα εξαχθούν απο το https://clinicaltrials.gov/.

## Τεχνολογίες που θα χρησιμοποιηθούν
* MySQL
* Javascript (nodejs, reactjs)
* HTML
* CSS
